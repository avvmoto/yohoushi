Yohoushi

Yohoushi is for all users who love graphs. It is a graphing tool designed to collaborate with GrowthForecast, Yohoushi as a front-end and GrowthForecast as backend.

Yohoushi shows you graphs stored in GrowthForecast in a multiple approach. You can see a large sized graph, specify start/end time date, and see a number of thumbnail graphs in one page. And it also supports an unlimited-layer hierarchical graph tree.

And it supports multiple GrowthForecast clusters. Yohoushi provides you highly scalable system.

If you are already an user of GrowthForecast, No reason not to try Yohoushi! Adding Yohoushi into the existing system is really easy, just run Yohoushi. You'll notice that you don't need to change the script to post a number to GrowthForecast because it refers to GrowthForecast.

Installation

Requirements

Install these first.

  1. GrowthForecast (>=0.62)
  2. Ruby (>=2.0.0)
  3. Rails (>=4.0.0)< /li>
  4. MySQL or Sqlite3 (>=3.6.16)

Note that Yohoushi uses MySQL as default. If you like sqlite3, override the configuration file cp config/database-sqlite.yml config/database.yml.

To install Yohoushi,

  1. git clone git@github.com:yohoushi/yohoushi.git
  2. cd yohoushi
  3. bundle install

Configuration

Open config/application.yml with an editor, and replace the url after :mapping syntax with the real url.

defaults: &defaults
  multiforecast:
  mapping:
    '':     'http://gfserver.local:5125'
  tagcloud:
  limit: 400

Run

Just,

$ bin/yohoushi

Note that you don't need bundle exec prefix.

And you're done! Point your web browser at http://localhost:4804
